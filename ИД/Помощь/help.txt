----- с 24.10.2011 ------

Дисциплина "Лингвистическое и программное обеспечение САПР" изучается по:
1. М.А.Песков, С.И.Борисов. Учебное пособие. "Лингвистическое  программное обеспечение САПР",-Томск: Факультет дистанционного обучения ТУСУР, 2010. - 108 с.

По дисциплине "Лингвистическое и программное обеспечение САПР" в течение семестра необходимо выполнить следующие работы:

  1. 2 текстовые лабораторные работы по заданию в учебном пособии.
    Лабораторная работа №1 состоит из 5 заданий по 6 вариантов в каждом (стр.77 ).
    Лабораторная работа №2 состоит из 1 задания по 6 вариантов (стр.84 ).
    Номер варианта выбирается по общим правилам.

Для лабораторной работы №1 Вы можете найти пример реализации модуля match (match.c + match.h) в архиве lab3vm.zip в папке match. Данный модуль готов для включения в проект, поэтому Вы можете использовать его непосредственно в таком виде.
Исходный текст алгоритма рекурсивного спуска для использования в лабораторной работе №1 Вы можете найти в архиве lab3vm.zip в папке recurs.
Пример реализации виртуальной машины для Лабораторной работы №2 Вы можете найти в архиве lab3vm.zip в папке vm.

Дополнительное ПО: Софт:\Cygwin

 При изучении дисциплины Вы можете пользоваться электронным учебным пособием, 
 расположенным в папке Tutor (Запускается файл Index.htm либо Start.htm либо TexBook.exe). 

 Также, по дисциплине предусмотрен электронный курс, который доступен на Сайте студента из учебного   плана ( в описании дисциплины "Методическое обеспечение)".
